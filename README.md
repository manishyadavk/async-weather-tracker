Async Weather Tracker 

A simple JavaScript Weather Application that fetches real-time weather data using the OpenWeatherMap API.
This project demonstrates the use of Asynchronous JavaScript (Async/Await, Fetch API) along with DOM manipulation, Local Storage, and Event Loop logging.


 Features:
 

 Search weather by city name

 Displays temperature, weather condition, humidity, and wind speed

 Search history stored using Local Storage

 Uses Async/Await with Fetch API

 Shows JavaScript Event Loop logs in console panel

 Simple and responsive UI using HTML & CSS
 

 Technologies Used:
 

HTML5

CSS3

JavaScript (ES6+)

OpenWeatherMap API

Local Storage
